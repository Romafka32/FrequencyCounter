Запуск приложения:
1.	Из папки build/libs скачать файл «FriquencyCounter-0.0.1-SNAPSHOT.jar». 
2.	Запустить файл на компьютере из командной строки с по команде “java -jar FriquencyCounter-0.0.1-SNAPSHOT.jar ”

Документация по формату I/O
1.	После запуска приложения в браузере перейти по ссылке http://localhost:8080/swagger-ui/index.html — откроется автоматическая документация Swagger. В ней будут описаны параметры входящих данных и ответов.
2.	В коде в формате Javadoc даны описания классов, методов и полей.
